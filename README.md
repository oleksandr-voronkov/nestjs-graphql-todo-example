## Description

An example of the Todo Application built with NestJS + Mongoose + GraphQl

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
